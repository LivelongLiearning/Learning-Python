# Learning-Python

> Структурирую и повторяю знания в летнее время.

## Цель проекта

Этот репозиторий — моя личная база знаний и площадка для отработки навыков Python. Основная задача на текущий период — систематизировать понимание ключевых концепций и закрепить их на практике.

## Фокус обучения

В данный момент я уделяю особое внимание следующим темам:

- **Контейнеры**: структуры данных, итераторы, генераторы, работа с коллекциями.
- **Git**: полный цикл работы с репозиторием, ветвление, разрешение конфликтов.
- **Объектно-ориентированное программирование (ООП)**: классы, наследование, инкапсуляция, полиморфизм.

## Как ведётся учёт прогресса

После каждого этапа редактирования в документацию будет добавляться информация о проделанной работе:

- Что было сделано (новые примеры кода, рефакторинг, тесты).
- Затраченное время (в часах).
